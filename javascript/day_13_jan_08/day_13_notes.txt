Javascript Runtime Env
======================
Heap : all the variables are stored physically
callstack : functions are aexecuted , LIFO 
            Normal functions are added to callstack directly
            async callbacks are not added to callstack directly

async functions have to wait in 
    1. Task Queue     (lower priority)  (settimeout, setinterval)
    2. MicroTask Queue (higher priority) (promise , async functions, mutation observer)
